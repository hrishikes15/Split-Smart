# SplitSmart Frontend Requirements

## Tech stack:
- React
- Tailwind CSS

## Design:
- Modern SaaS-style UI
- Clean and minimal
- Dark theme preferred

## Features:

**1. Layout:**
- Sidebar navigation
- Top header
- Main dashboard area

**Sidebar:**
- Dashboard
- Groups
- Expenses
- Settlements

**2. Dashboard:**
- Stats cards:
  - Total expenses
  - Total owed
  - Total to receive
  - Settled amount

**3. Groups:**
- Create group
- View members
- Group overview

**4. Expenses:**
- Add expense modal
- Choose:
  - paid by
  - split type (equal/custom)
- Show expense list

**5. Balances:**
- Show each user’s net balance
- Indicate:
  - owes money
  - gets money

**6. Settlement View:**
- Display optimized transactions:
  - "A pays B ₹500"
- Mark transactions as settled

**7. UI Details:**
- clean cards
- smooth hover effects
- loading states
- empty states

**8. API Integration:**
- connect with backend APIs
- fetch groups, expenses, balances, settlements

**9. Code Quality:**
- reusable components
- clean folder structure
- separate UI and API logic
